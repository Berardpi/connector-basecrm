===== The aim of the connector is to synchronise Maestrano Connec! API with BaseCRM API. 


- SSO with Maestrano  
- OAuth2 handshake with Base
- Synchronizations successful
- Data Shared updating correctly after Synch with BaseCRM for:
        - People and Organizations ----------> Contacts
        - Items                    ----------> Producs
